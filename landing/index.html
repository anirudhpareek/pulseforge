What it ships

Everything needed to go from demo motion to production shipping.

  • Single-canvas render loop tuned for streaming data.
  • Shape-preserving decimation for high-frequency feeds.
  • Storybook, tests, benchmark harness, and stress scenarios.
  • TypeScript-first package with ESM/CJS outputs.
  • Dark-first visual defaults with easy theme overrides.

Design principles

Calm by default

Animation clarifies state change, never distracts from value change.

Legibility under pressure

Dense streams remain readable through aggressive but faithful decimation.

Operationally real

Built with stress and benchmark tooling, not visual demos alone.

PulseForge is for teams building data interfaces where movement and trust both matter. It is ideal for trading surfaces, monitoring dashboards, ops tooling, and any product where realtime context should feel smooth and intentional.

The package ships with practical guardrails: reduced-motion support, SSR-safe rendering, and built-in test + benchmark workflows to avoid regressions as datasets and feature scope grow. 

import { PulseForge } from '@anirudhpareek/pulseforge';

<PulseForge
  mode="line"
  lineData={stream}
  width={900}
  height={320}
  maxPoints={700}
  showFps
/>
