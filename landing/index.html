Realtime charts for modern products

0 fps 0 points line mode

PulseForge is a React canvas package for realtime charts. It combines Liveline-inspired movement with practical engineering: high-frequency decimation, line and candlestick modes, SSR-safe behavior, reduced-motion handling, and test + benchmark tooling.

The goal is simple: keep motion expressive but never noisy, keep data dense but never unreadable, and keep implementation straightforward enough to ship in serious products. 

import { PulseForge } from '@anirudhpareek/pulseforge';

<PulseForge
  mode="line"
  lineData={stream}
  width={900}
  height={320}
  maxPoints={700}
  showFps
/>
