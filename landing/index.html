Realtime charting for modern products

Ship streaming interfaces that feel alive and stay readable.

PulseForge is a React canvas chart engine for high-frequency data. It gives you smooth motion, line and candlestick modes, and practical engineering defaults for production use.

0 fps 0 points line mode

Why teams use it

Most libraries look fine in static screenshots and degrade in realtime. PulseForge is tuned for moving data first: decimate aggressively, preserve shape, and keep transitions informative instead of decorative.

  • ● Realtime loop on a single canvas surface
  • ● Shape-preserving decimation for heavy streams
  • ● SSR-safe behavior and reduced-motion awareness
  • ● Storybook demos, tests, benchmark and stress tooling

Quick usage

One component, predictable props, production defaults.

import { PulseForge } from '@anirudhpareek/pulseforge';

<PulseForge
  mode="line"
  lineData={stream}
  width={900}
  height={320}
  maxPoints={700}
  showFps
/>
