PulseForge is a React chart package built for streaming interfaces: smooth motion, readable data density, and reliable behavior under load.

0 fps 0 points line mode

Built by Anirudh Pareek · March 2026

Why teams use PulseForge

Most chart libraries degrade when data updates continuously. PulseForge keeps interfaces readable by combining shape-preserving decimation, a focused canvas render loop, and animation that explains change instead of adding noise.

  • Single-canvas realtime rendering for lower overhead
  • Line and candlestick modes with consistent behavior
  • Reduced-motion and SSR-safe defaults
  • Stress harness and benchmark scripts for operational confidence

Quick usage

Drop in one component and tune max points for your stream density.

import { PulseForge } from '@anirudhpareek/pulseforge';

<PulseForge
  mode="line"
  lineData={stream}
  width={900}
  height={320}
  maxPoints={700}
  showFps
/>
