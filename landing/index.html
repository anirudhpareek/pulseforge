Introducing PulseForge: realtime chart motion that stays readable under load.

AP

Anirudh Pareek

Product + Engineering

4 min read Mar 3, 2026

PulseForge is a React canvas chart package inspired by Liveline-style motion. It keeps streaming data visually calm, ships with line and candlestick modes, and includes decimation + stress tooling so performance remains predictable.

0 fps 0 points line mode

The core idea

Most chart libraries get noisy when streams get fast. PulseForge favors ordering and legibility first: decimate aggressively, preserve shape, and animate transitions only when they improve comprehension.

What ships in the package

  • Single-canvas render loop with low overhead
  • Line and candlestick rendering modes
  • Decimation for high-frequency feeds
  • SSR-safe fallback and reduced-motion support
  • Storybook examples, tests, benchmark and stress harness

Quick usage

import { PulseForge } from '@anirudhpareek/pulseforge';

<PulseForge
  mode="line"
  lineData={stream}
  width={900}
  height={320}
  maxPoints={700}
  showFps
/>
